# Riddhi Jain – Portfolio

Personal portfolio: Full Stack Developer & Data Analyst. Built with Next.js 14, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Push to GitHub, then import the repo on [Vercel](https://vercel.com) and deploy (defaults are fine for Next.js).

## Structure

- `app/` – layout, page, global styles
- `components/` – sections (Hero, About, Skills, Experience, Projects, Education, Contact, Footer)
- `public/` – resume, certificates, project screenshots, decoration images
